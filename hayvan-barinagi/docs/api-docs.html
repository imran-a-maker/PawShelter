PawShelter API

RESTful API dokümantasyonu. Tüm endpoint'ler JSON formatında veri alır ve döner.

Base: https://api.pawshelter.com/api/v1 Content-Type: application/json Auth: Bearer JWT

Authentication

POST /auth/register Yeni kullanıcı kaydı

Yeni bir kullanıcı hesabı oluşturur. Başarılı kayıt sonrası JWT token çifti döner.

Request Body
{
  "name": "Ayşe Kaya",
  "email": "ayse@example.com",
  "password": "SecurePass123!",
  "phone": "05551234567"
}
Response 201
{
  "access_token": "eyJhbGci...",
  "refresh_token": "eyJhbGci...",
  "token_type": "bearer",
  "user": {
    "id": 42,
    "name": "Ayşe Kaya",
    "email": "ayse@example.com",
    "role": "user"
  }
}
POST /auth/login Kullanıcı girişi

Email ve şifre ile giriş yapar, JWT token çifti döner.

Request Body
{
  "email": "ayse@example.com",
  "password": "SecurePass123!"
}
Response 200
{
  "access_token": "eyJhbGci...",
  "refresh_token": "eyJhbGci...",
  "token_type": "bearer"
}

PUT /animals/{id} Hayvan güncelle Admin

Belirtilen ID'ye sahip hayvanın bilgilerini günceller. Kısmi güncelleme (PATCH) de desteklenir.

DELETE /animals/{id} Hayvan sil Admin

Hayvanı sistemden siler. Aktif sahiplenme talebi varsa 409 Conflict döner.

Adoptions

POST /adoptions/request Sahiplenme talebi oluştur Auth

Kimliği doğrulanmış kullanıcı bir hayvan için sahiplenme talebi oluşturur.

Request Body
{
  "animal_id": 15,
  "message": "Evimiz büyük, bahçemiz var..."
}
Response 201
{
  "id": 101,
  "animal_id": 15,
  "user_id": 42,
  "status": "pending",
  "created_at": "2024-05-28T..."
}
Hata Durumları
409 AnimalNotAvailableError – Hayvan sahiplenilebilir durumda değil
409 DuplicateRequestError – Aktif talep zaten mevcut
404 AnimalNotFoundError – Hayvan bulunamadı
GET /adoptions/requests Talepleri listele Admin

Tüm sahiplenme taleplerini listeler. status, user_id, animal_id ile filtrelenebilir.

PUT /adoptions/requests/{id} Talebi onayla / reddet Admin

Response 200

  "status": "approved",

Kod Örnekleri

services/adoption_service.py 
# services/adoption_service.py
from typing import Optional
from datetime import datetime
from domain.entities import AdoptionRequest, Animal, User
from domain.enums import AdoptionStatus, AnimalStatus
from repositories.interfaces import IAdoptionRepository, IAnimalRepository
from core.exceptions import (
    AnimalNotAvailableError,
    DuplicateRequestError,
    AdoptionNotFoundError
)

class AdoptionService:
    """
    Sahiplenme iş mantığını yöneten servis katmanı.
    Single Responsibility Principle: Yalnızca sahiplenme operasyonlarından sorumlu.
    """

    def __init__(
        self,
        adoption_repo: IAdoptionRepository,
        animal_repo: IAnimalRepository
    ):
        # Dependency Inversion: Concrete sınıflar yerine interface'e bağımlı
        self._adoption_repo = adoption_repo
        self._animal_repo = animal_repo

    async def create_adoption_request(
        self,
        user_id: int,
        animal_id: int,
        message: Optional[str] = None
    ) -> AdoptionRequest:
        """Yeni bir sahiplenme talebi oluşturur."""
        # 1. Hayvanın varlığını ve uygunluğunu kontrol et
        animal = await self._animal_repo.get_by_id(animal_id)
        if animal.status != AnimalStatus.AVAILABLE:
            raise AnimalNotAvailableError(
                f"Hayvan ID {animal_id} sahiplenilebilir durumda değil. "
                f"Mevcut durum: {animal.status.value}"
            )

        # 2. Mükerrer talep kontrolü
        existing = await self._adoption_repo.find_pending_request(
            user_id=user_id, animal_id=animal_id
        )
        if existing:
            raise DuplicateRequestError(
                f"Kullanıcı {user_id} için bekleyen talep zaten mevcut."
            )

        # 3. Yeni talep oluştur
        new_request = AdoptionRequest(
            user_id=user_id,
            animal_id=animal_id,
            message=message,
            status=AdoptionStatus.PENDING,
            created_at=datetime.utcnow()
        )

        # 4. Hayvanı "Beklemede" durumuna güncelle
        await self._animal_repo.update_status(
            animal_id=animal_id, new_status=AnimalStatus.PENDING
        )

        # 5. Talebi kaydet ve döndür
        return await self._adoption_repo.create(new_request)

    async def approve_adoption_request(
        self, request_id: int, admin_id: int
    ) -> AdoptionRequest:
        """Admin tarafından sahiplenme talebini onaylar. Atomik işlem."""
        async with self._adoption_repo.transaction():
            request = await self._adoption_repo.get_by_id(request_id)
            if not request:
                raise AdoptionNotFoundError(f"Talep ID {request_id} bulunamadı.")

            request.status = AdoptionStatus.APPROVED
            request.reviewed_by = admin_id
            request.reviewed_at = datetime.utcnow()
            await self._adoption_repo.update(request)

            # Hayvanı "Sahiplendirildi" olarak işaretle (Atomik)
            await self._animal_repo.update_status(
                animal_id=request.animal_id, new_status=AnimalStatus.ADOPTED
            )

            # Diğer bekleyen talepleri reddet
            await self._adoption_repo.reject_other_pending_requests(
                animal_id=request.animal_id, approved_request_id=request_id
            )

        return request
